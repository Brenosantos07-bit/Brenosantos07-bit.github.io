Central de API

Descrição

Este projeto consistiu no desenvolvimento de uma Central de API para a SEFAZ, funcionando como um gateway para gerenciar, proteger e documentar os diversos microsserviços e endpoints de dados da empresa. A plataforma oferece autenticação segura, monitoramento de requisições e uma interface para que outros desenvolvedores possam consumir os dados de forma controlada.
