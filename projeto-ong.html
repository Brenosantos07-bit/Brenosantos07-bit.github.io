Site ONG

Descrição

Este projeto envolveu o desenvolvimento front-end completo para uma Organização Não Governamental focada na preservação ambiental. O objetivo era criar um site informativo, atrativo e de fácil navegação para divulgar as causas da ONG, atrair voluntários e captar doações.

Tecnologias Utilizadas

HTML5 CSS3 JavaScript
