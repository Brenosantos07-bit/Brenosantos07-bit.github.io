App AjuBus

Descrição

O AjuBus é um aplicativo web progressivo (PWA) desenvolvido para facilitar o uso do transporte público pela população de Aracaju. O app permite que os usuários visualizem rotas de ônibus em tempo real, consultem horários e planejem suas viagens de forma eficiente, diminuindo o tempo de espera nos pontos.

Pretendo desenvolver esse projeto mais a frente, usando HTML, CSS E JS.
