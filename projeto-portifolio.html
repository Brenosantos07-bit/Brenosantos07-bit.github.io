Descrição

Este projeto consistiu no desenvolvimento de umEste projeto é o meu próprio portfólio profissional, desenvolvido do zero para servir como uma vitrine central para minhas habilidades e trabalhos. O objetivo foi criar uma experiência de usuário limpa, intuitiva e totalmente responsiva, que funcionasse perfeitamente em qualquer dispositivo. Partindo de um design concebido no Figma, construí cada seção página por página, utilizando HTML5 semântico para a estrutura, CSS3 com Flexbox e Grid para layouts complexos, e JavaScript puro (Vanilla JS) para adicionar interatividade, como o carrossel de projetos e a validação do formulário de contato. O resultado é o site que você está navegando agora.
